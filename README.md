# The Subway Surfers

Play Subway Surfers online without delay.

## Features

- **Instant Play** - One click to start playing
- **Berlin Map** - Classic Berlin edition
- **Fullscreen Mode** - Immersive gameplay experience
- **Responsive** - Works on desktop and mobile

## Tech Stack

- Vue 3 (Composition API)
- Vue Router 4
- Vite 7

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Ready for Vercel deployment with `vercel.json` configured.
